# Brain games

- Even. Answer "yes" if even. Otherwise, answer "no".
- Calculator. Correctly add, subtract or multiply.
- GCD. Greatest common divisor.
- Progression. Find a pattern.
- Prime. Answer "yes" if natural. Otherwise, "no".

## How to Run
```sh
   make run-dist
```
